## BUNNINGS - In Store

### Timber & Panels
- [ ] 1/2" (12mm) Baltic birch plywood sheets - for furring strips (~$30-50/sheet, need 2-3 sheets)
- [ ] 3mm project panels 1220x2440mm - walls (need 6-8 sheets @ ~$30-50 each) = **~$240-400**
- [ ] 6mm plywood sheets - ceiling (need 2-3 sheets @ ~$40-60 each) = **~$120-180**
- [ ] Pine 2x2 or 1x2 DAR - alternative furring option (per meter, calculate based on van ribs)
- [ ] Pine 2x2 or 2x3 - for overhead cabinet frames if doing stick-built

### Hardware - Fasteners
- [ ] M6 or M8 bolts (20-30mm length) - for furring to rivnuts (qty: 3-4 per strip x number of strips)
- [ ] M6 or M8 washers (matching bolts)
- [ ] Lath screws 30mm (1 1/4") - 500g box for panels
- [ ] Wood screws 25-40mm pan head - for panels/cabinets
- [ ] Self-tapping screws 12-14mm - backup option
- [ ] Gas struts - for overhead cabinets if lift-up design ($20-40 per pair)

### Tools & Bits
- [ ] Countersink bit or Forstner bit set (for recessing bolt heads)
- [ ] Jigsaw blades (metal cutting for roof, wood for panels)
- [ ] Sandpaper assortment (80, 120, 240 grit)

### Adhesives & Sealants
- [ ] Sikaflex 221 or 291 (marine sealant) - for MaxxAir, roof penetrations, shore power ($25-35/tube)
- [ ] Marine grade construction adhesive - optional for panels ($10-15)
- [ ] Timber glue - for cabinets
- [ ] Contact adhesive - if vinyl wrapping panels

### Electrical
- [ ] 2.5mm² twin & earth cable (TPS) - for shore power (electrician to specify length) (~$3-5/meter)
- [ ] Cable clips/saddles - for AC wiring
- [ ] Cable ties - various sizes
- [ ] Electrical tape

### Plumbing (if not buying specialty)
- [ ] Hose clamps - various sizes for water system
- [ ] PVC or flexible hose - depends on system design

### General Supplies
- [ ] Stainless steel screws - various sizes
- [ ] Spray paint (touch-up for rust) - Ford Transit color match
- [ ] Black bumper paint/restorer
- [ ] Wood stain/varnish (optional for panels)
- [ ] Drop sheets, rags, cleaning supplies

**Estimated Bunnings Total: $600-900** (depends on timber quantities)

## PURCHASING STRATEGY

### Week 1: Measurements & Planning
1. Take all van measurements
2. Calculate exact quantities for timber, wire, etc.
3. Scout Marketplace for IKEA parts, water tanks, tools

### Week 2: Phase 1 Purchases
1. Buy MaxxAir fan (online order, allow shipping time)
2. Buy sealant and roof work supplies

### Week 3-4: Phase 2 Bulk Purchase
1. Big Bunnings run: All timber, hardware, fasteners
2. Jaycar run: All 12V lighting and wiring
3. Buy IKEA bed components

### Week 5-6: Phase 3 & 4 Prep
1. Shore power components (if doing this early)
2. Water system parts (CaravansPlus/RV Parts online order)
3. Lagun table system order

### Week 7-8: Phase 5 & 6
1. Fabric for window covers
2. Roof rack (may need to order)
3. Cabinet hardware from Bunnings
4. Roof storage boxes

### Ongoing: Marketplace Monitoring
- Check daily for second-hand bargains
- Snap up good deals on IKEA, tools, etc.
- Be patient for rare items (stainless tanks, Lagun tables)

---

## PRIORITY ITEMS TO BUY FIRST

1. **Van measurements tools** (tape measure, spirit level) - if you don't have
2. **MaxxAir fan** - long lead time, order early
3. **Furring strip timber** - Phase 2 depends on this
4. **12V wiring and lights** - order/buy before panelling starts
5. **Water pump & tanks** - may have lead time, research options early
